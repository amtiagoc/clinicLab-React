## How to run it:

1. Clone the repository

```sh
$ git clone https://github.com/cano2030/Frontend_clinicLab.git
$ cd Frontend_clinicLab
```

2. Install dependencies:

```sh
$ npm install
```

3. Run the server:

```sh
$ npm start
```
