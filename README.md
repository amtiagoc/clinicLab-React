## How to run it:

1. Clone the repository

```sh
$ git clone https://github.com/cano2030/clinicLab-React.git
$ cd clinicLab-React
```

2. Install dependencies:

```sh
$ npm install
```

3. Run the server: *Keep in mind that you have to be running the backend to get the information
required.*

```sh
$ npm start
```
